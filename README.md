# Overview

The backend for NindoPhx: an 90% feature complete rewrite of [Nindo](https://github.com/RobinBoers/Nindo) in Elixir.

## Features

- Markdown support
- Comments
- Dark mode
- Readable
- RSS feeds
- REST API
- Responsive and quick
